写好 System Prompt

三个技巧，大幅提升智能体响应质量与一致性

TIP 01
角色设定要具体
不要写"你是一个助手"，要写"你是一位有 10 年 XX 经验的..."。具体的身份会带来更聚焦的输出，减少无关内容。
TIP 02
明确输出格式
在 prompt 中直接规定输出结构（如：必须包含结论、分析、建议三部分），可以大幅提升响应质量和一致性。
TIP 03
设定行为边界
告诉智能体什么情况下要主动说"信息不足"，避免凭空捏造数据。结论先行，数据支撑，不做无依据的乐观预测。

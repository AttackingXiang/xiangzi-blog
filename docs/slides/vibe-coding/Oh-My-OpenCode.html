安装与配置

两分钟快速上手

支持两种安装方式，推荐直接在 AI 助手会话中一键完成配置，无需手动操作。

1

AI 助手一键安装 推荐

在 Cursor / Claude Code / OpenCode 会话中直接粘贴安装提示词，AI 自动完成全部配置。

Install and configure oh-my-opencode by following the instructions here:
https://raw.githubusercontent.com/code-yeongyu/oh-my-opencode/...
2

交互式命令行安装

通过 Bun 或 npx 在终端中交互式完成安装，过程中准备好你的 API Key 即可。

# 使用 Bun（推荐，速度极快） bunx oh-my-opencode install # 或使用 npx npx oh-my-opencode install

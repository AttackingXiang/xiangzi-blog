# Claude Code 插件总览

> 以下插件均来自官方插件市场 `anthropics/claude-plugins-official`。
>
> 使用任何插件前，需先执行一次：
> ```bash
> /plugin marketplace add anthropics/claude-plugins-official
> ```

---

## 全部插件一览

| 插件名称 | 安装量 | 核心功能 | 安装指令 | 触发命令 |
|---------|:------:|---------|---------|---------|
| [frontend-design](./claude-plugins/frontend-design) | 413.6K | 生成高质量、有设计感的前端 UI，告别千篇一律的 AI 风格 | [`/plugin install frontend-design@claude-plugins-official`](./claude-plugins/frontend-design) | 自动激活，无需命令 |
| [superpowers](./claude-plugins/superpowers) | 294.8K | 超能力模式：头脑风暴 + 子代理协作 + 多步骤规划 | [`/plugin install superpowers@claude-plugins-official`](./claude-plugins/superpowers) | [`/superpowers <任务>`](./claude-plugins/superpowers) |
| [context7](./claude-plugins/context7) | 212.2K | 实时注入第三方库最新文档，解决训练数据过时问题 | [`/plugin install context7@claude-plugins-official`](./claude-plugins/context7) | [`use context7`（在提问中加入）](./claude-plugins/context7) |
| [code-review](./claude-plugins/code-review) | 191.9K | 四个并行 Agent 全维度审查 PR，附置信度评分 | [`/plugin install code-review@claude-plugins-official`](./claude-plugins/code-review) | [`/code-review`](./claude-plugins/code-review) |
| [code-simplifier](./claude-plugins/code-simplifier) | 159.9K | 自动简化重构代码，提升可读性，功能不变 | [`/plugin install code-simplifier@claude-plugins-official`](./claude-plugins/code-simplifier) | [`/code-simplifier <文件>`](./claude-plugins/code-simplifier) |
| [github](./claude-plugins/github) | 158.3K | 官方 GitHub MCP 服务器，直接操作仓库 / PR / Issue | [`/plugin install github@claude-plugins-official`](./claude-plugins/github) | 自然语言描述 GitHub 操作 |
| [feature-dev](./claude-plugins/feature-dev) | 143.9K | 七阶段功能开发工作流，三个专属 Agent 协作完成 | [`/plugin install feature-dev@claude-plugins-official`](./claude-plugins/feature-dev) | [`/feature-dev <需求>`](./claude-plugins/feature-dev) |
| [playwright](./claude-plugins/playwright) | 134.6K | 浏览器自动化与 E2E 测试 MCP 服务器 | [`/plugin install playwright@claude-plugins-official`](./claude-plugins/playwright) | [`/playwright <任务>`](./claude-plugins/playwright) |
| [ralph-loop](./claude-plugins/ralph-loop) | 120.6K | 自引用 AI 闭环迭代，持续执行直到满足完成条件 | [`/plugin install ralph-loop@claude-plugins-official`](./claude-plugins/ralph-loop) | [`/ralph-loop "<任务>" --completion-promise "<条件>"`](./claude-plugins/ralph-loop) |
| [typescript-lsp](./claude-plugins/typescript-lsp) | 116.5K | TypeScript/JS 语言服务器：类型检查 + 代码补全 + 定义跳转 | [`/plugin install typescript-lsp@claude-plugins-official`](./claude-plugins/typescript-lsp) | 自动激活（.ts/.tsx/.js/.jsx 文件） |
| [claude-md-management](./claude-plugins/claude-md-management) | 110.2K | 维护优化 CLAUDE.md，保持项目文档与代码同步 | [`/plugin install claude-md-management@claude-plugins-official`](./claude-plugins/claude-md-management) | [`/revise-claude-md`](./claude-plugins/claude-md-management) |
| [skill-creator](./claude-plugins/skill-creator) | 105.9K | 创建和优化 Claude 自定义技能，封装复用工作流 | [`/plugin install skill-creator@claude-plugins-official`](./claude-plugins/skill-creator) | [`/skill-creator <技能描述>`](./claude-plugins/skill-creator) |

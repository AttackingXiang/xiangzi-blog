在 Mac mini 终端上操作（接着显示器键盘）：

## 第一步：编辑网络配置文件



bash

```bash
nano /etc/network/interfaces
```

## 第二步：找到这段内容并修改

现在文件里大概是这样：



```
auto vmbr0
iface vmbr0 inet static
        address 192.168.1.100/24
        gateway 192.168.1.1
        bridge-ports nic0
        bridge-stp off
        bridge-fd 0
```

把 `address` 和 `gateway` 改成新的：



```
auto vmbr0
iface vmbr0 inet static
        address 192.168.10.4/24
        gateway 192.168.10.1
        bridge-ports nic0
        bridge-stp off
        bridge-fd 0
```

> ⚠️ `gateway` 填你新路由器的 IP，一般是 `192.168.10.1`，如果你的路由器管理地址不是这个请替换成实际的。

**保存退出**：`Ctrl+O` → 回车 → `Ctrl+X`

## 第三步：同步修改 DNS（可选但建议）



bash

```bash
nano /etc/resolv.conf
```

把里面的 `nameserver 192.168.1.1` 改成：



```
nameserver 192.168.10.1
nameserver 223.5.5.5
```

保存退出同上。

## 第四步：重启网络生效



bash

```bash
systemctl restart networking
```

## 第五步：验证



bash

```bash
ip a
```

看 `vmbr0` 那行，`inet` 后面应该变成 `192.168.10.4/24`，并且状态从 `DOWN` 变成 `UP`（前提是新路由器网线已经插好）。

再 ping 一下新路由器：



bash

```bash
ping -c 3 192.168.10.1
```

通了就成功，然后 Windows 浏览器访问 `https://192.168.10.4:8006` 登录后台。

------

**操作顺序建议**：先改完配置保存，再把网线插到新路由器上，最后执行 `systemctl restart networking`，这样一步到位。

在电脑上下载一个软件，github地址：

```
https://github.com/Ackites/Nrfr
```

选最近的版本，下载

![image-20260412200411875](%E4%BF%AE%E6%94%B9sim%E5%8D%A1%E5%9B%BD%E5%AE%B6%E7%A0%81.assets/image-20260412200411875.png)

1、手机进入设置，点击系统版本7下（小米系统），进入开发者选项



2、退出设置到主页面，找到更多设置，里边会多一个开发者选项



3、打开“USB调试”，打开“USB安装”，打开“USB调试（安全设置）"



然后手机连接电脑

打开github下载的软件

解压后运行

![image-20260412202526675](%E4%BF%AE%E6%94%B9sim%E5%8D%A1%E5%9B%BD%E5%AE%B6%E7%A0%81.assets/image-20260412202526675.png)

选择设备

![image-20260412200923803](%E4%BF%AE%E6%94%B9sim%E5%8D%A1%E5%9B%BD%E5%AE%B6%E7%A0%81.assets/image-20260412200923803.png)

提示，会在手机上安装这两个应用，点击安装

![image-20260412200942119](%E4%BF%AE%E6%94%B9sim%E5%8D%A1%E5%9B%BD%E5%AE%B6%E7%A0%81.assets/image-20260412200942119.png)

![image-20260412201003435](%E4%BF%AE%E6%94%B9sim%E5%8D%A1%E5%9B%BD%E5%AE%B6%E7%A0%81.assets/image-20260412201003435.png)

此时手机会提示要安装应用，选择允许



点击确认并启动，

![image-20260412201054059](%E4%BF%AE%E6%94%B9sim%E5%8D%A1%E5%9B%BD%E5%AE%B6%E7%A0%81.assets/image-20260412201054059.png)

提示，“现在可以开始使用了”。代表电脑部分安装成功了

![image-20260412201138119](%E4%BF%AE%E6%94%B9sim%E5%8D%A1%E5%9B%BD%E5%AE%B6%E7%A0%81.assets/image-20260412201138119.png)



手机这里的shizuku第一行，等待一下会提示正在运行

<img src="%E4%BF%AE%E6%94%B9sim%E5%8D%A1%E5%9B%BD%E5%AE%B6%E7%A0%81.assets/image-20260412201540101.png" alt="image-20260412201540101" style="zoom:50%;" />

打开另一个软件nrfr

选择你要变的sim卡，选择国家码，选择运营商（运营商随便选择一个）

<img src="%E4%BF%AE%E6%94%B9sim%E5%8D%A1%E5%9B%BD%E5%AE%B6%E7%A0%81.assets/image-20260412201606746.png" alt="image-20260412201606746" style="zoom:50%;" />

点击保存生效，修改完成，可以尝试tiktok应用了
